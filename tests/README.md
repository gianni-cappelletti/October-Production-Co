# Test Fixtures

Shared test data (WAV files) used by all three test suites:
- `libs/octobir-core/tests/` -- core library tests
- `plugins/juce-multiformat/tests/` -- JUCE plugin tests
- `plugins/vcv-rack/tests/` -- VCV Rack plugin tests

Defined as `TEST_DATA_DIR` in each suite's `CMakeLists.txt`.
